# DomoticzHttp
An example alarm demo based on Arduino and Domoticz. Json API and the HTTP protocol were used to communicate with the Domoticz platform.
## Used
* Arduino Uno Rev3
* Arduino Ethernet Shield 2
* 3 x DS18B20
* DHT11
